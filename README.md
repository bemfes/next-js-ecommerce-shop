# 🛍️ Интернет-Магазин на Next.js

Современное и адаптивное веб-приложение интернет-магазина, разработанное с использованием **Next.js**. Пользователи могут просматривать каталог товаров, фильтровать их по различным параметрам, открывать страницы отдельных товаров, управлять корзиной и оплачивать покупки с помощью **Stripe**.

🌐 [Демо проекта](https://next-js-shop.netlify.app/)

## 🛠️ Стек

* Next.js
* TypeScript
* Zustand
* Tailwind CSS
* Shadcn UI 
* Stripe

## 🎯 Функционал

* Просмотр каталога товаров
* Страница для каждого товара
* Фильтрация товаров
* Добавление товаров в корзину
* Удаление товаров из корзины
* SEO-данные для каждой страницы
* Загрузка каталога товаров из Stripe API

## 🚀 Установка и запуск

1. Клонируйте репозиторий

```
https://github.com/bemfes/next-js-ecommerce-shop.git
```

2. Установите зависимости
```
npm install
```

3. Добавьте в корень проекта .env файл с переменными:
```
STRIPE_SECRET_KEY = YOUR_STRIPE_SECRET_KEY

NEXT_PUBLIC_BASE_URL = 'http://localhost:3000'
```
❗ У вас должен быть Stripe аккаунт с загруженными товарами

4. Запустите проект в режиме разработки
```
npm run dev
```
